# ✨ PixelErase AI

AI-powered background removal web application built with Flask, modern backend architecture, and real-time image processing.

---

# 🚀 Features

- 🎯 AI Background Removal
- 🖼 Multi Image Upload Support
- ⚡ Real-Time Processing
- 🎨 Custom Background Selection
- 🌈 Background Color Options
- 📥 Instant Download
- 🧠 Modular Flask Backend
- 🗂 Scalable Project Architecture
- 💾 Database Integration with SQLAlchemy
- 🔥 REST API Support
- 📱 Responsive Modern UI

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/Shubhi-glitch/pixelerase-ai.git
```

---

## Open Project

```bash
cd pixelerase-ai
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

---

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Project

```bash
python run.py
```
